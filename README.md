# Canvas

*  版本：1.1.1
*  作者：InFaNg

## 使用
1. 下载并在`<head>`中引入Canvas文件
    <link rel="stylesheet" href="canvas.css">
2. 为想要添加动画效果的元素赋予以下一个类（具体效果可参考[Canvas 文档](http://infang.github.io/Canvas/)）
  * `appear`
  * `disappear`
  * `appear-top`
  * `disappear-top`
  * `appear-right`
  * `disappear-right`
  * `appear-bottom`
  * `disappear-bottom`
  * `appear-left`
  * `disappear-left`
  * `appear-rotate`
  * `disappear-rotate`
  * `fast-appear`
  * `fast-disappear`
  * `fast-appear-top`
  * `fast-disappear-top`
  * `fast-appear-right`
  * `fast-disappear-right`
  * `fast-appear-bottom`
  * `fast-disappear-bottom`
  * `fast-appear-left`
  * `fast-disappear-left`
  * `fast-appear-rotate`
  * `fast-disappear-rotate`
  * `shake-x`
  * `shake-y`
  * `flash`
  * `rotation-clockwise`
  * `appear`
